# FP01 – Formulários em PHP

## Descrição
Este projeto consiste em exercícios práticos de **processamento de dados via formulário** em PHP, realizados no VS Code. O trabalho incluiu:  
- **Saudação personalizada:** Recebe o nome do utilizador via formulário e apresenta uma mensagem de boas-vindas.  
- **Classificação etária:** Recebe a idade do utilizador e devolve a classificação correspondente (Criança, Adolescente ou Adulto).  
- **Nível de password:** Avalia a força da password inserida com base no tamanho e apresenta o resultado (Fraca, Média, Forte).

## Como Executar
1. Colocar a pasta `m5-fp01` dentro do diretório `htdocs` do XAMPP.  
2. Iniciar o servidor Apache no XAMPP.  
3. Aceder via browser a `http://a14714-oficina.infinityfree.me/m5/form.html` (ou aos outros formulários correspondentes).
